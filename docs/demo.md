# OJ Demo — Interactive Prototype

OJ includes a live, interactive prototype built using Lovable.

👉 **Launch the prototype:** https://oj-wfm.lovable.app/

**Repo alignment:** how this demo fits next to docs and mocks is summarised in [`CANONICAL.md`](CANONICAL.md).

## What the demo shows
- The OJ Wheel  
- Segment interactions  
- Task flows  
- Mobile-first UI  
- Visual identity  
- Completion mechanic (“drink your juice”)  

## Why this matters
The prototype brings the OJ concept to life and demonstrates how the product could feel in a real environment.
